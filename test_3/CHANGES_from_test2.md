# test_2 → test_3 변경사항 · 변경 이유

> 요약: test_2의 "MinerU 사전파싱 → LLM 전사" 기조는 유지하되, **검색 코어(청크색인+리랭커)** 와
> **신뢰성 계층(검증·롤백·CRAG)**, **비전 오독 회피(도표 텍스트화)** 를 더해 **챗봇 투입 가능 수준**으로
> 끌어올렸다. 이후 **Phase 6에서 생성·검증 LLM을 Gemini로 실험**(rag3x). 근거:
> `결과물/probes_results/FINAL_REPORT.md`, `PHASE6_COMPARISON.md`, `결과물/rag3_참고_실험팩트정리.md`.

## 1. 왜 바꿨나 (동기)

test_2는 빠르고 수치는 정확했지만 **챗봇 모듈로 쓰기엔** 결함이 있었다:

- **vision 오독 86%**(스캔표·도표가 비전 경로로 뜨면 오독) · **근거 있어도 실패 33%** ·
  **page_hit@3 55.6%**(페이지 통짜 색인의 검색 천장) · **무검증 단방향**(틀린 답을 못 잡음).

→ "검색을 정밀화하고, 답을 **검증·롤백**하며, **도표는 비전 대신 텍스트로** 다루자."

## 2. 핵심 변경 (Before → After)

| 항목 | test_2 | test_3 (rag3) | 이유 / 근거 |
|---|---|---|---|
| 색인 단위 | 페이지 통짜 | **MinerU 블록 청크**(969p→~2,417청크) | 검색 정밀도 |
| 리랭킹 | 없음 | **`bge-reranker-v2-m3` 크로스인코더** | page-level +8%p(Phase 0) |
| 벡터DB | Chroma | **numpy+BM25 flat 인덱스** | Chroma 대형컬렉션 크로스프로세스 로드 실패(B6) 원천 제거 |
| 카탈로그 게이트 | 사용 | **제거**(리랭크 floor + 메타 프리픽스 주입) | 13문서 소규모에선 게이트가 손해(doc_hit 0.692 vs 0.923) |
| 검증 | **없음** | **controller S7 검증**(숫자 결정론 대조 + groundedness + is_abstain) | 환각 0 |
| 롤백/재검색 | 없음 | **S8 결정론 롤백 3종 + CRAG 질의재작성** | 실패 자기교정 |

| 오케스트레이션 | 단일 함수 | **controller 상태기계 S1~S8** | 검증·롤백 루프 |

## 3. 무엇이 좋아졌나 (효과, 36문항)

| 지표 | test_2 | test_3 |
|---|---|---|
| page_hit@3 | 0.556 | **0.774** |
| doc_hit@3 | 0.923 | **1.0** |
| vision 오독률 | 86% | **0%** |

- 부수 효과: 도표 텍스트화가 **검색 앵커**를 만들어 page_hit@3을 0.645→0.774로 사후 상승시킴(Phase 1 목표 달성의 핵심 요인).
- 후반 신뢰성 이슈 규명: **12b 콜드 whitespace 런어웨이**(콜드 KV에서 공백 폭주→답변 잘림)를
  `retry_on_length`로 방어 → avg_kw_hit +8.5%p.

## 4. Phase 6 — Gemini 실험 (rag3x, 별도 병행 포크)

- **동기**: 로컬 12b는 호출당 10~15s×호출수가 시간의 95% → "전 문항 20초"는 API로만 가능. 다문서 종합 강화.
- **범위**: **생성·검증 LLM 자리만** Gemini `gemini-3.1-flash-lite`로 치환. **임베딩·리랭커·검색은 로컬 유지**
  (재색인·종속 방지). 기존 `rag3` 코드는 **한 줄도 수정 안 함**(전 플래그 OFF ⇒ 등가).
- **결과(C)**: kw 0.528→**0.719**, 지연 63s→**6.2s**(전 문항 <20s), 파싱천장 4/5 개방, 비용 ~$0.0004/문항,
  환각 0·무관거절 100% 유지. 종합형 문서간은 D(분해+문장인용)가 0.625→0.917.
- ⛔ **대체 / 유지 / 병행 결정은 사용자 몫**(`PHASE6_COMPARISON.md` A/B/C/D 비교표 기준). Claude가 채택 결정 안 함.
